# 📈 Super Store Sales & Profitability Analysis (Power BI)

## Project Overview
This project involves a comprehensive analysis of global retail data (2011-2014) to identify key drivers of revenue and profitability. I built an interactive Power BI dashboard to help stakeholders visualize sales performance across regions and understand how discounting strategies affect the bottom line.

## Key Insights & Features
* **Sales & Profitability Overview:** Analyzed 51K total orders with a focus on a 0.08 profit margin against a 0.20 target.
* **Decomposition Tree Analysis:** Utilized AI-driven visuals to break down the highest and lowest revenue-generating markets and regions.
* **Discount Impact Study:** Discovered that heavy discounts in categories like Tables and Bookcases are significantly eroding margins.
* **Customer Segmentation:** Identified the Top 10 customers by revenue to facilitate targeted loyalty programs.
* **Interactive Slicers:** Enabled deep-dives by Region, Category, and Year (2011–2014).

## Technical Skills Used
* **Power BI:** Data Visualization, DAX Measures, and Dashboard Design.
* **Power Query:** Data transformation and cleaning of global sales records.
* **Business Analytics:** KPI tracking, trend analysis, and margin optimization strategy.
